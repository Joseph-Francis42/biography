# Michael Jackson Biography Page

A premium, highly aesthetic static biography page celebrating the career milestones, achievements, and legacy of the King of Pop, Michael Jackson. Built entirely with clean semantic HTML5 and vanilla CSS3.

## Features

- **Sleek Gold-on-Black Design**: Styled with deep charcoal glassmorphism cards and bright gold highlights.
- **Key Milestones Timeline**: A vertical, responsive chronology listing the major eras of his life and career.
- **Interactive Links**: Smooth-scrolling navigation anchors.
- **Facts Grid**: Compact card overview of born/died details, record achievements, and Grammy awards.
- **Inspirational Quotes**: Grid section showcasing some of his most impactful statements.
- **Responsive Layout**: Adapts seamlessly from wide desktop monitors to mobile screens.

## Files

- [index.html](index.html): Document markup containing semantic sections and metadata.
- [styles.css](styles.css): Stylesheet containing design tokens, HSL color schemes, custom scrollbars, and responsiveness.
- [mj_portrait.png](mj_portrait.png): Stylized silhouette graphic representing the King of Pop.

## How to Run

1. **Directly**: Simply open the `index.html` file in any modern web browser.
2. **Local Server**: Run a lightweight server inside this directory:
   ```bash
   npx http-server -p 8080
   ```
   Then open `http://localhost:8080` in your web browser.
